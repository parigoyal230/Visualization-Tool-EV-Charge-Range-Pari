**Project Summary:**  
This project analyzes electric vehicle charging patterns using Tableau.  
- Data: Connection time, charging duration, kWh delivered, day of week.  
- Estimated Range: kWh × 5 km/kWh (average EV efficiency).  
- Visualizations: Avg kWh per day, peak charging times, duration vs energy scatter.  
- Insights: Evening peaks, weekday patterns, better range planning for India.  

**Tools:** Tableau for dashboard & story, Kaggle dataset.
